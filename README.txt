How to deploy (quick):
1) GitHub Pages:
   - Create a new GitHub repo, upload files at root, enable Pages to serve from main branch root.
   - Add custom domain: www.BavlyAlber.com and follow GitHub instructions for DNS.
2) Netlify (drag & drop):
   - Sign in to Netlify and drag the folder or ZIP to Sites -> New site from drag & drop.
   - Add custom domain in Site settings and follow DNS instructions.
DNS:
- In domain provider, create CNAME for 'www' pointing to your Netlify/GitHub Pages host.
- Wait for propagation. SSL can be enabled from host dashboard.
Notes:
- Replace YouTube VIDEO_ID placeholders with your real video IDs.
- Replace Formspree yourFormID with your form ID or use Google Forms.
- Replace map iframe links with project-specific embeds.
